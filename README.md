# 一、选题背景与意义
《密阵突围》是综艺节目《最强大脑》第13季中的经典挑战项目，其核心玩法基于代码矩阵的联动合并机制。玩家需在5a×5a的网格中选择a个“操纵代码”，通过上下左右移动实现与相邻代码的合并，合并规则涉及字母的五进制加法与数字的个位保留，且合并后的新格子自动成为操纵代码，操纵代码总数动态减少，极具逻辑性与策略性。

本课题旨在利用C++面向对象编程思想，完整还原《密阵突围》的核心玩法，并通过文件操作实现题库管理、成绩持久化及对局回放功能。该项目不仅能够巩固C++语法、STL及文件I/O知识，还能锻炼复杂逻辑规则的工程化实现能力，是一次理论与实践结合的良好机会。

二、研究目标与内容
2.1 总体目标
开发一款基于C++的《密阵突围》单机游戏程序，实现完整的合并规则、可变盘面大小、动态操纵代码管理、用户答题记录及排行榜功能，并支持选做的题目自定义与对局回放。
2.2 具体内容
(1) 核心游戏逻辑
参数a输入：游戏开始前，提示用户输入a值（正整数，取值范围1~10），盘面大小为5a×5a，初始操纵代码数量为a。
盘面初始化：生成5a×5a的网格，每个格子包含随机代码（字母A-E，数字0-9）。
操纵代码选择：玩家在盘面上选择a个格子作为初始操纵代码。
  1.选择方式：键盘输入坐标或图形界面点击
  2.验证：选中数量必须等于a，否则无法开始游戏
联动移动：所有选中的操纵代码同时响应移动指令（上/下/左/右）。
合并判定与执行：
  1.移动前，对每个操纵代码检查其目标格子的状态
a.情况一：目标格子为空→操纵代码直接移动到目标格，不计入合并。
    b.情况二：目标格子有代码→进行合并判定：
    (1)若字母相同：数字相加保留个位，字母不变
    (2)若数字相同：字母按五进制相加（A=0,B=1,C=2,D=3,E=4），结果映射回A-E，数字不变
    (3)若字母数字均相同：保持不变
    (4)若均不相同：移动失败，操纵代码停留原处
  2.合并后处理：原操纵代码位置清空，合并后的新格子自动成为操纵代码
  3.特殊情况：若多个操纵代码的目标冲突，按规则解决（详见技术方案）
操纵代码数量变化：每次合并后，被合并的目标格成为新的操纵代码，而原操纵代码位置被清空
终止判定：当所有操纵代码都无法移动或合并时，游戏结束
得分计算：记录成功合并的次数（不含纯移动）

(2) 交互与界面
基础版本：控制台界面，使用字符网格显示盘面，键盘WASD控制移动，支持坐标输入选择初始代码。
图形化版本（选做/加分）：采用QT框架实现图形化界面，包含网格显示、按钮控制、状态提示。

(3) 题库与数据持久化
内置至少3道预设题目（不同a值）
支持用户选择题号开始游戏
文件操作：不使用数据库，所有数据存储于文本文件
  questions.txt：存储题目（a值+盘面初始状态）
  scores.txt：存储用户成绩（玩家名、a值、用时、合并次数、得分率）
  replay.txt：存储每步移动序列

(4) 用户排行榜
记录玩家用时与合并次数，计算得分率（合并次数/总操作次数或理论最大合并数）
支持按总分或按不同a值分别排名
支持查看排行榜

(5) 扩展功能
自定义题目：允许用户编辑盘面并保存到题库
对局回放：将每一步操作写入文件，支持逐步回放
自动出题：随机生成合法且有一定难度的盘面
难度分级：a值越大难度越高，可作为困难模式
